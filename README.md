# Customer Orders Analysis (SQL Project)

## Project Description
This project focuses on analyzing customer order data using SQL. The goal is to retrieve meaningful insights such as total orders, customer activity, and sales trends.

## Tools Used
- MySQL 8.0 Command Line Client

## Database Structure
Table: orders  
Columns:
- order_id
- customer_name
- order_date
- order_amount

## SQL Skills Demonstrated
- SELECT queries
- WHERE filtering
- ORDER BY
- Aggregate functions (SUM, COUNT)
- GROUP BY

## Key Insights
- Total orders and revenue were calculated.
- Customer purchase behavior was analyzed.
- Order data was filtered and sorted for reporting.

## Outcome
This project demonstrates the ability to analyze transactional data using SQL and generate business-focused insights suitable for entry-level data and SQL freelancing tasks.
